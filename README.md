# SLanguage

## 简介

SLanguage 是一种简单易用的编程语言，旨在为开发者提供一个简洁而高效的编程环境。它的语法简洁，易于学习和使用，同时保持了足够的灵活性，以满足各种编程需求。

## 安装

### 系统要求

- 操作系统：Windows / Linux
- 内存：至少 16MB
- 磁盘空间：至少 20MB

### 安装步骤

1. 下载最新版本的 SLanguage 解释器。
2. 解压缩下载的文件。
3. 将 SLanguage 编译器添加到系统路径中。
